# disable-hover
Disable hover effects on scroll

##Install

```
$ bower i disable-hover
```

```html
<html>
<head>
  ...
  <link rel="stylesheet" href="bower_components/disable-hover/dist/disable-hover.css">
</head>
<body>
  ...
  <script src="bower_components/disable-hover/dist/disable-hover.min.js">
</body>
</html>
```
